An end-to-end machine learning web application for predicting customer churn, explaining prediction drivers, visualizing analytics, and processing batch predictions from CSV uploads.

---

## Overview

The **Customer Churn Intelligence Platform** helps businesses identify customers who are likely to leave, understand the reasons behind churn risk, and make better retention decisions.

This project combines:

- a trained machine learning model
- a FastAPI backend
- a modern responsive frontend
- SHAP-based explainability
- analytics dashboards
- batch prediction with CSV upload

---

## Features

### Single Customer Prediction
Predict whether a customer is likely to churn using key customer behavior and subscription features.

### Explainable AI
SHAP explanations show the most important drivers behind each prediction.

### Retention Recommendation
The system provides simple business guidance based on predicted churn risk.

### Prediction History
All predictions are stored and displayed through a history page.

### Analytics Dashboard
Interactive charts summarize churn trends, risk levels, and subscription distributions.

### Batch Prediction
Upload a CSV file to predict churn for multiple customers at once and download results.

### Dark Mode
A modern dark mode interface improves user experience.
